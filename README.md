# MyBudget
<p>A user-friendly personal budget tracker app!</p>


## Description of innovative features

### Innovative Feature 1: Unique functionality

<p>MyBudget includes records used to create single income or expense instances and must have both a category and an account assigned to them. Categories are used to easily sort records. Accounts allow visualisation of different money holdings the user has access to such as cash, different cards or online wallets.</p>

<p>MyBudget uses common, region-specific formulas to calculate goals that must be met.</p>


### Innovative Feature 2: Unique user experience

<p>MyBudget includes graphs, a calculator when storing records, visual indicators and allows users to export their data.</p>

<p>MyBudget includes pictures on loading screens and icons on each object.</p>
